Welcome to the BootBoard API

This is a RESTful API for managing users,boards, and entries using basic CRUD operations

Available endpoints:

  • GET /bootboard/api/v1/users - List all users - connecting to MySQL database
  • GET /bootboard/api/v2/users - List all users - using Redis caching layer
  • GET /api/users/{id} - Get a user by ID
  • POST /api/users - Create a new user
  • PUT /api/users/{id} - Update a user
  • DELETE /api/users/{id} - Delete a user
